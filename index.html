CNAME dosyası nedir?
GitHub Pages’e “bu repo şu domain ile çalışsın” demek için GitHub otomatik/manuel olarak repo köküne CNAME dosyası koyar. İçinde sadece şunun gibi bir satır olur:
turkuntoresi.com
Şu an 404 görürsen:
1) Sayfa linki doğru mu? → kök adresi aç: /
2) DNS oturması 10 dk – birkaç saat sürebilir (bazen 24 saate uzar).
3) GitHub Pages “Custom domain / DNS check” yeşile dönünce HTTPS seçeneği aktif olur.
